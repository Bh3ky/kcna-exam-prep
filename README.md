# KCNA Exam Prep

A collection of study materials for the Kubernetes and Cloud Native Associate (KCNA) certification based on the Linux Foundation’s Kubernetes and Cloud Native Essentials (LS250) course.

## Contents

* Structured study notes
* Practice questions
* Mock examinations
* Flashcards
* Diagrams and visual aids
* Curated learning resources

## Exam Domains

* Kubernetes Fundamentals
* Container Orchestration
* Cloud Native Architecture
* Cloud Native Observability
* Cloud Native Application Delivery
* Cloud Native Networking
* Cloud Native Security

## Purpose

This repository serves as a personal knowledge base and revision guide while preparing for the KCNA certification.
